Ticker Tycoon

  • React Native
  • iOS
  • Android
  • Rails

Ticker Tycoon is a paper trading app created in React Native. User authentication and profile data is facilitated through a Rails API. User sessions are stored in LocalStorage. Charts are drawn with Highcharts, and data is sourced from Yahoo Finance and Google Finance.

Poker

  • Javascript
  • Canvas
  • EaselJS
  • HTML/CSS

Poker is a single-player poker game based on the 5 card draw rules. Features include a Fisher-Yates shuffling algorithm to ensure random hands, as well as shuffle and card flip/draw animations with EaselJS.

Algorithm Racer

  • Rails
  • React
  • Codemirror Editor
  • Node VM

Algorithm Racer allows users to test search and sort algorithms in Javascript. It uses the Codemirror Editor for syntax highlighting, and runs user-submitted code safely in a Node VM instance. Submitted algorithms are scored on speed and brevity and added to a leaderboard.
